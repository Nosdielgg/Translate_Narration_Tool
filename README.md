
Translate_Narration_Tool _ Translate &amp; Narration Tool  Python desktop app for translation and text-to-speech narration. 
Features - Translate SRT, DOCX, TXT → DOCX - Generate MP3 narration from text or SRT - Neural voices (Microsoft Edge TTS) - 
6 languages supported  - Installation: pip install edge-tts deep-translator python-docx python srt_tts.py
